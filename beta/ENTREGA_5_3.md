# Entrega 5.3 — Reemplazo completo del catálogo

- La importación acepta archivos XLSX.
- La hoja Productos se reemplaza completamente por el catálogo importado.
- Ya no se calculan productos nuevos ni cambios de nombre o precio.
- Los códigos se conservan como texto, incluidos los ceros iniciales.
- Se validan todas las filas antes de escribir.
- Se eliminan duplicados exactos dentro del archivo, conservando la última aparición.
- La hoja Stock no se modifica.
- Las filas sobrantes del catálogo anterior se limpian después de escribir correctamente el catálogo nuevo.
